01 / SELECTED WORK

Lumina Video Engine

WebGLReactRust

The Narrative

Lumina was born from a frustration with modern video editors: they are either too simple for pros or too complex for creators. We built a custom engine that handles 8K raw footage directly in the browser using WebAssembly.

The challenge was maintaining 60fps performance while applying complex color grading matrices in real-time. We achieved this by offloading heavy frame processing to a custom Rust-based worker pool.

DESIGN ENGINEERING WORKFLOW

How I Build

01

Discovery & Logic

I start by mapping out the technical constraints and the underlying logic of the design. This involves defining state machines, data structures, and the mathematical foundations of the interface.

  • State Architecture
  • Design System Audit
  • Performance Budgeting
02

Prototyping in Code

Static mocks only tell half the story. I build high-fidelity interactive prototypes in the target language (React, WebGL, or Swift) to feel the physics, the motion, and the actual user friction.

  • Component Sandboxing
  • Motion & Physics Tuning
  • Multi-modal Testing
03

Production Deployment

The final step is translating the prototype into clean, maintainable, and type-safe production code. I focus on accessibility, bundle size optimization, and visual regression testing.

  • Semantic HTML/A11y
  • CI/CD Visual Testing
  • Asset Optimization
